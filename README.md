# なぞってコネクト

同じ色のピースをなぞって消していく、楽しいブラウザパズルゲームだ！

まずは同じ色を3個以上つなげて消そう。慣れてきたら5個消しで特殊ブロックを作り、盤面を一気に動かして高得点を狙え！特殊ブロックを仕込んで、つないで、5色レインボーを作ろう！

## 基本ルール

- 同じ色のピースを3個以上なぞると消せます。
- 斜めにもつなげられます。
- 5個以上つなげると特殊ブロックを生成できます。
- 同じ個数の通常ブロック消しを続けると REN が伸びます。
- 何も消さずに指を離しただけでは REN は途切れません。

## 特殊ブロック

特殊ブロックは、このゲームの主役です。普通に消すだけでは届かない大技を狙えます。

使い道は大きく5つあります。

1. 盤面を動かす
  特殊ブロックを短くタッチすると移動モードになります。そのあと動かしたいピースをタッチすると、そのピースが特殊ブロックの場所へ移動し、空いた場所は落下で埋まります。あと1個足りない形を作るときに強いです。
2. 5個消しで特殊ブロックを増やす
  同じ色を5個以上つなげると、消したピースの1つが特殊ブロックになります。まずはここから始めると、盤面がどんどん派手になります。
3. 特殊ブロック込みの同色消しで色をまとめて消す
  同じ色の通常ブロック2個と特殊ブロックをつなげると、その色の通常ブロックをまとめて消せます。合計5個以上なら、新しい特殊ブロックも作れます。盤面整理と次の大技づくりを同時にできます。
4. 同じ色の特殊ブロックをつなげてライン生成
  同じ色の特殊ブロック同士をつなぐと、通った方向のラインが同じ色の通常ブロックに変わります。次の5個消しや連鎖の土台を作れる、仕込み型の強技です。
5. 違う色の特殊ブロックを混ぜて大爆発
  違う色の特殊ブロックをつなげると、広い範囲をまとめて破壊できます。色を多く混ぜるほど強く、全色を混ぜるとレインボーボーナスに届きます。

特殊技を連続させると BTB が入り、得点がさらに伸びます。次の特殊技繋げろ！

## レインボーボーナス

特殊ブロックを全色混ぜると、最強のレインボーボーナスが発動します。

レインボーは大量破壊に加えて時間を30秒増やせます。つまり、点が入る、盤面が吹き飛ぶ、時間も増える、全部乗せの最強技です。

高ランクを目指すなら、ただ3個ずつ消すよりも、特殊ブロックを温存して全色ミックスまで持っていくのが近道です。

## GitHub Pages

このプロジェクトは GitHub Actions から GitHub Pages へデプロイします。

1. GitHub の repository settings で Pages の source を `GitHub Actions` にする。
2. `main` ブランチへ push する。
3. `.github/workflows/deploy.yml` が Astro build を実行し、Pages へ公開する。

公開 URL は標準設定では `https://neko-kurage.github.io/Nazotte-Connect/` です。別アカウントや organization で公開する場合は [astro.config.mjs](./astro.config.mjs) の `site` を変更してください。

## バージョン公開

最新版は常に `https://neko-kurage.github.io/Nazotte-Connect/` で公開します。

過去版は Git tag を使って固定します。たとえば `v0.1.0` タグを push すると、GitHub Actions がそのタグ時点のゲームを `https://neko-kurage.github.io/Nazotte-Connect/v0.1.0/` に同梱して公開します。

```bash
git tag v0.1.0
git push origin v0.1.0
```

新しいバージョンを公開したら、README の Versions 表へリンクを追加してください。

ローカルで過去版のパスを確認する場合は、対応する dev script を使います。

```bash
npm run dev:v0.1.0
```

別の dev server が動いている場合は、先に停止してから起動し直してください。

```bash
npx astro dev stop
```

`package.json` の version が `0.1.0` なら、確認 URL は `http://localhost:4321/Nazotte-Connect/v0.1.0/` です。
